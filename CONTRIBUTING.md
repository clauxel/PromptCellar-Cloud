# Contributing

Thanks for improving the PromptCellar Cloud docs.

## Rules

- Keep changes public-safe.
- Use concise summaries instead of copying long external passages.
- Add UTM tracking to every public HTTP link.
- Do not include credentials, private logs, private repository details, customer data, or local machine paths.
- Prefer small, reviewable pull requests.

## Link Check

Before publishing, search for bare HTTP links and confirm they use UTM unless they are internal anchors, mail links, or localhost development notes.
