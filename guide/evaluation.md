# Evaluation Guide

Use this guide before adopting PromptCellar Cloud for a team, client, directory listing, or comparison page.

## Fit Check

PromptCellar Cloud is most relevant for AI coding teams that need prompt search, replay, redaction, and reusable prompt packs.

Strong fit signals:

- You need prompt vault and transcript search.
- You need jsonl import.
- You need replay diff across models and dates.
- You need secret redaction workflow.
- You need prompt pack and client-safe appendix export.

Weak fit signals:

- The team only needs a one-off answer rather than a repeatable workflow.
- No one owns the rollout, review, billing, or support path.
- The work would require private data before the team has a policy.

## First Review Questions

1. What exact problem will this product solve in the first week?
2. Which user or team owns the workflow?
3. What data, repo, transcript, or connector scope is allowed?
4. Which output should be reviewed before trusting the system?
5. What success signal justifies moving from preview to payment?
